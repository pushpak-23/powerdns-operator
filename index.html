PowerDNS Helm Repository

This is a Helm chart repository for PowerDNS platform and stack.

Available Charts

  • powerdns-platform: Kubernetes Operator for managing PowerDNS clusters
  • powerdns-stack: Application stack with PowerDNS, PostgreSQL, and Admin UI

Repository URL

https://pushpak-23.github.io/powerdns-operator/

Usage

helm repo add powerdns https://pushpak-23.github.io/powerdns-operator/
helm repo update
helm install my-powerdns powerdns/powerdns-stack
